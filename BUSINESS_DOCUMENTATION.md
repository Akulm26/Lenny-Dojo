## 1. Executive Summary

**Lenny's Dojo** is an AI-powered product management interview preparation platform that transforms insights from Lenny Rachitsky's renowned podcast into actionable practice questions and personalized coaching.

### Mission Statement

> "To democratize access to world-class product management wisdom by converting expert insights into structured, AI-coached interview preparation experiences."

### Key Metrics

| Metric | Value |
|--------|-------|
| Podcast Episodes Processed | 303+ |
| Companies Analyzed | 180+ |
| Frameworks Extracted | 80+ |
| Interview Types Covered | 9 |
| Evaluation Dimensions | 5 |
| AI Model | Google Gemini 3 Flash Preview / 2.5 Pro (+ BYOK support) |

### Key Value Propositions

- **Real Context**: Interview questions derived from actual product decisions at top companies
- **AI Coaching**: Personalized evaluation with detailed feedback on 5 key dimensions
- **Expert Frameworks**: Curated library from industry leaders
- **Company-Specific Prep**: Targeted preparation for specific company interviews
- **Progress Analytics**: Track improvement with actionable insights

## 2. Problem Statement

### The PM Interview Challenge

Product management interviews are notoriously difficult to prepare for. Unlike technical interviews with clear right/wrong answers, PM interviews require demonstrating:

- Nuanced thinking
- Strategic frameworks
- Real-world decision-making skills
- Communication excellence
- Business acumen

### Current Pain Points

| Pain Point | Impact |
|------------|--------|
| Generic practice questions lack context | Low confidence in answers |
| No feedback mechanism for practice answers | No improvement loop |
| Scattered resources across multiple platforms | Wasted preparation time |
| Difficulty simulating real interview pressure | Poor time management |
| No structured progression tracking | Unclear readiness |

### Lenny's Dojo Solution

| Problem | Solution |
|---------|----------|
| Generic questions | Questions from real company decisions |
| No feedback | AI coaching with dimensional feedback |
| Scattered resources | Centralized, curated content library |
| No pressure simulation | Timer modes for pressure simulation |
| No tracking | Comprehensive progress analytics |

### Market Opportunity

- **300,000+** product managers in the US
- **2-3 year** average tenure (continuous interview demand)
- High willingness to invest for FAANG/top-tier positions
- Growing PM job market with 15%+ annual growth

## 3. Solution Overview

Lenny's Dojo leverages a unique content pipeline that extracts actionable intelligence from Lenny's Podcast—the #1 product management podcast—and transforms it into structured interview preparation experiences.

### Intelligence Extraction Pipeline

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│    YouTube      │    │  AI Processing  │    │  Structured     │    │   Practice      │
│   Transcripts   │ -> │  Gemini 2.5 Pro │ -> │     Data        │ -> │   Questions     │
│  (Auto-synced)  │    │                 │    │                 │    │ (Model Answers) │
└─────────────────┘    └─────────────────┘    └─────────────────┘    └─────────────────┘
```

### Core Platform Components

| Component | Description |
|-----------|-------------|
| 🏢 **Company Intelligence** | Deep dives into product decisions at top companies |
| 📚 **Framework Library** | Curated frameworks from industry experts |
| 🎯 **Practice Sessions** | Configurable mock interviews with AI evaluation |
| 📈 **Progress Tracking** | Performance analytics across dimensions |

### Data Extraction Per Episode

From each podcast episode, the system extracts:

- **Companies Mentioned**: Names, decisions, outcomes
- **Frameworks Discussed**: Methodology, use cases, examples
- **Expert Opinions**: Quotes, context, insights
- **Practice Questions**: 3-5 questions with model answers per episode

Every practice answer is evaluated across 5 key dimensions:

| Dimension | Weight | What It Measures |

| Framework Usage | 20% | Appropriate application of PM frameworks |

| Outcome Orientation | 20% | Focus on results and impact |

| Option | Questions | Duration |

| Standard | 3 | ~30 min |
